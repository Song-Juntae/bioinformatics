### 뇌파란 무엇인가?

뇌에는 약 1000억개 뉴런이 있고, 각 뉴런은 다른 뉴런과 약 1000번 이상의 시냅스를 이룬다.
뉴런에 임계치 이상 입력이 들어오면 Action potential이 발생, 입력이 커지면 커질수록 Action potential 발생 횟수 증가
